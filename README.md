# 🦷 Sistema DentAgenda - Sistema de Gestión de Citas Odontológica

Sistema de control de citas e historiales médicos dentales, diseñado para el Consultorio Odontológico L. Orellano F.P.

## ✨ Características Clave
*   Registro de pacientes.
*   Agenda de citas.
*   Historial clínico digital con odontograma.
*   Generación de reportes básicos.

## ⚠️ Estado del Proyecto: Prototipo 20
**Este software se encuentra en fase de desarrollo.** Es una versión temprana, por lo que puede contener errores o funciones incompletas. No se recomienda su uso en entornos de producción sin una validación exhaustiva.

## 🚀 Instalación
1.  Descarga el instalador desde la sección [Releases](https://github.com/Carlos-IUTEPAL/Dent-Agenda-Aplicacion-/releases) de este repositorio.
2.  Ejecuta el archivo `Sistema DentAgenda Instalador.exe` e instala el programa en tu equipo.
3.  Sigue las instrucciones del asistente de instalación.
